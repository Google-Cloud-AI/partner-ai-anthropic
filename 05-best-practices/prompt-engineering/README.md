# Prompt Engineering

> ⚠️ **Use at your own risk.** See [root disclaimer](../../README.md).

## Overview
Practical prompting guidance for Claude: structuring prompts, using clear instructions and examples, leveraging XML tags, and encouraging step-by-step reasoning.

## Prerequisites
- Basic familiarity with calling Claude

## What You'll Learn / What This Demonstrates
- Write clear, detailed instructions
- Use positive and negative examples effectively
- Structure prompts with XML tags for reliable parsing
- Apply techniques for longer-context and reasoning tasks

## Quick Start
```bash
# This module is guidance-only — see the patterns and
# annotated examples in this README and accompanying notes.
```

## Architecture
Not applicable — this is a guidance module with example prompts, not a deployed system.

## Cost Considerations
Better prompting reduces wasted tokens and retries, which directly lowers per-token spend.
