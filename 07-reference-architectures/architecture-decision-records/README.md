# Architecture Decision Records

> ⚠️ **Use at your own risk.** See [root disclaimer](../../README.md).

This folder holds Architecture Decision Records (ADRs). Each ADR documents a single significant decision: its context, the decision made, and its consequences.

Number ADRs sequentially, e.g. `0001-use-vertex-ai-multi-region.md`. A common starting template is [Michael Nygard's ADR format](https://github.com/joelparkerhenderson/architecture-decision-record).
